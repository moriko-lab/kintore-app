# PUMP

自分専用の筋トレ記録 PWA。広告なし、余計な機能なし。iOS ネイティブ準拠のデザイン（リポジトリ名は kintore-app のまま）。

## 機能

- カレンダー起点の記録（月表示、記録日にドット）
- 種目・重量・回数・セットの記録
- 前回コピー（前回セッションのセット内容を複製）
- セット間タイマー（完了タップで自動起動、秒数は設定で変更可）
- 部位別の最終トレーニング経過日数
- 種目ごとの最大重量推移グラフ
- JSON / markdown エクスポート（iOS 共有シート対応、iCloud Drive 等へ保存）
- JSON インポート（復元用）
- オフライン動作（Service Worker）

## 技術

- 素の HTML / CSS / JavaScript（ビルドなし、依存なし）
- データは IndexedDB（端末内）に保存。サーバーなし
- GitHub Pages で配信

## 使い方（iPhone）

1. Safari で公開 URL を開く
2. 共有ボタンから「ホーム画面に追加」
3. ホーム画面のアイコンから起動（スタンドアロン表示）

データは端末のブラウザ内にのみ保存される。ホーム画面のアイコンを削除するとデータも消えるため、設定画面から定期的に JSON を書き出して iCloud 等へ保存すること。
